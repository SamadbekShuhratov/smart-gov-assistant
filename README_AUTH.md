Global state\nconst [authToken, setAuthToken] = useState(\"\");\nconst [authUsername, setAuthUsername] = useState(\"\");\nconst [showAuthForm, setShowAuthForm] = useState(false);\n\n// On mount: Restore from localStorage\nuseEffect(() => {\n  const token = localStorage.getItem(\"authToken\");\n  const username = localStorage.getItem(\"authUsername\");\n  if (token && username) {\n    setAuthToken(token);\n    setAuthUsername(username);\n  }\n}, []);\n\n// On login: Save to localStorage and close form\nconst handleLoginSuccess = ({ token, username }) => {\n  setAuthToken(token);\n  setAuthUsername(username);\n  localStorage.setItem(\"authToken\", token);\n  localStorage.setItem(\"authUsername\", username);\n  setShowAuthForm(false);\n};\n\n// On logout: Clear everything\nconst handleLogout = () => {\n  setAuthToken(\"\");\n  setAuthUsername(\"\");\n  localStorage.removeItem(\"authToken\");\n  localStorage.removeItem(\"authUsername\");\n  setShowAuthForm(false);\n};\n```\n\n---\n\n## 🎨 UI Components\n\n### Modal Overlay\n```jsx\n{!authToken && (\n  <div className=\"fixed inset-0 z-40 flex items-center justify-center bg-black/50 backdrop-blur-sm\">\n    {/* Welcome modal with Login/Register buttons */}\n  </div>\n)}\n```\n\n### Content Blur/Disable\n```jsx\n<main className={`space-y-4 ${!authToken ? \"pointer-events-none opacity-40 blur-sm\" : \"\"}`}>\n  {/* Main app content */}\n</main>\n```\n\n### Auth Form Panel\n```jsx\n{showAuthForm && (\n  <div id=\"auth-section\">\n    <AuthForm onLoginSuccess={handleLoginSuccess} activeTab={authTabTarget} />\n  </div>\n)}\n```\n\n---\n\n## 📱 Responsive Design\n\n- ✅ Mobile optimized (< 640px)\n- ✅ Tablet responsive (640px - Use httpOnly cookies instead of localStorage\n2. Implement CSRF token protection\n3. Add token expiration (15-30 minutes)\n4. Implement token refresh logic\n5. Hash passwords with bcrypt\n6. Add rate limiting on auth endpoints\n7. Implement 2FA (optional)\n8. Add security headers\n9. Use HTTPS only\n10. Regular security audits\n\nSee [NEXT_STEPS.md](./NEXT_STEPS.md) for security roadmap.\n\n---\n\n## 📈 Performance\n\n### Build Metrics\n- **JavaScript**: 182 KB (56 KB gzipped)\n- **CSS**: 27.11 KB (5.90 KB gzipped)\n- **HTML**: 0.40 KB (0.27 KB gzipped)\n- **Total Gzipped**: ~62 KB\n\n### Optimizations\n- ✅ Code splitting by route\n- ✅ CSS-based blur (GPU accelerated)\n- ✅ Efficient React rendering\n- ✅ No unnecessary re-renders\n- ✅ localStorage (instant reads)\n- ✅ Smooth 60fps transitions\n\n---\n\n## 🐛 Known Issues & Limitations\n\n### Current\n- Profile page not yet implemented (button is placeholder)\n- Auth button labels not localized\n- In-memory data (no database)\n- No email verification\n- No password reset flow\n\n### Planned Fixes\nSee [NEXT_STEPS.md](./NEXT_STEPS.md) for full roadmap.\n\n---\n\n## 🤝 Contributing\n\n### Local Development\n```bash\n# Install dependencies\ncd frontend && npm install\ncd backend && pip install -r requirements.txt\n\n# Start both servers\n# Terminal 1:\ncd backend && .venv/bin/uvicorn app.main:app --reload\n\n# Terminal 2:\ncd frontend && npm run dev\n```\n\n### Code Style\n- Use Prettier for formatting\n- Follow ESLint rules\n- Write meaningful commit messages\n- Add tests for new features\n\n---\n\n## 📞 Support\n\n### Resources\n- React Docs: https://react.dev\n- FastAPI Docs: https://fastapi.tiangolo.com\n- Tailwind CSS: https://tailwindcss.com\n- JWT: https://jwt.io\n\n### Getting Help\n1.
